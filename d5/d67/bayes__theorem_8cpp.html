Detailed Description

Bayes' theorem

Bayes' theorem allows one to find \(P(A|B)\) given \(P(B|A)\) or \(P(B|A)\) given \(P(A|B)\) and \(P(A)\) and \(P(B)\).
Note that \(P(A|B)\) is read 'The probability of A given that the event B has occured'.
